# Pro Investor & Swing Trader Analysis Template

Use this to produce a blunt, first-principles analysis for a single stock. Replace variables in brackets.

Inputs
- Ticker: [TICKER]
- Exchange: [NSE|BSE|US]
- Lookback: [DAYS]

Sections
1) Stock Classification
- Choose ONE: Long-term compounder | Cyclical | Momentum | Thematic/story-driven | Mean-reversion
- Why it fits; why it does not fit the others.

2) Business Quality Check
- ROCE, ROE over time; Capital intensity; Debt profile; Pricing power.
- State if structurally capable of compounding.
- Typical expert constraints: ROCE ≥ 15%, ROE ≥ 15%, D/E ≤ 0.5, stable margins.

3) Stock Behavior Analysis
- Triggers for rallies; causes for drawdowns; role of news/policy/sentiment.
- Separate business fundamentals from price behavior.

4) Suitable Strategy (pick ONE)
- Long-term hold | Swing trade only | Momentum trade | Avoid for now
- Justify choice clearly.

5) How to Use This Stock
- If tradable: Ideal holding period; Entry conditions; Exit signals.
- If long-term: What to track quarterly; When to add; When to exit.
- Technical definitions: RSI(14), SMA50/200, ATR(14), 52-week high.

6) Risks and Invalidations
- Top 3 risks; What data invalidates the thesis.

7) Investor Type Fit
- Who should own it; Who should not.

Notes
- If fundamentals are unavailable, state unknown and avoid guessing.
- For NSE/BSE, use Yahoo tickers with suffixes: `.NS` for NSE, `.BO` for BSE.
